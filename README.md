# Monster Adventure

A browser-based adventure game built with vanilla JavaScript and HTML5 Canvas.

## Overview

Navigate through multiple levels, defeat monsters, and progress through an exciting adventure. Features pixel art graphics inspired by classic games and custom typography.

## Features

- **Multi-level gameplay** - Progress through 3+ levels with increasing difficulty
- **Player character** - Control your character with keyboard input
- **Monster encounters** - Face various monsters with different behaviors
- **Canvas-based rendering** - Smooth 2D graphics using HTML5 Canvas API
- **Keyboard controls** - Responsive keyboard input handling
- **Custom assets** - Themed fonts and organized sprite management

### Controls

- **Arrow Keys (Left & Right)** - Move character
- **Space/Up Arrow Key** - Jump

## Languages Ised

- **HTML** - Page structure
- **CSS** - Basic Styling and layout
- **JavaScript** - Game engine
